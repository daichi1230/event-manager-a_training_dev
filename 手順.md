# Event Manager A コピペ手順

## 1. ローカルで起動する

### PowerShell
```powershell
cd <このフォルダを置いた場所>\event-manager-a
npm start
```

開く URL:
```text
http://localhost:3000
```

> もし 3000 番ポートが他で使われているなら、次で起動します。

```powershell
$env:PORT=3210
npm start
```

開く URL:
```text
http://localhost:3210
```

## 2. GitHub に push する

### 2-1. GitHub で新しい repository を作る
例: `event-manager-a`

### 2-2. ローカルから push する
```powershell
cd <このフォルダを置いた場所>\event-manager-a
git init
git add .
git commit -m "Initial commit for Event Manager A"
git branch -M main
git remote add origin https://github.com/<GitHubユーザー名>/event-manager-a.git
git push -u origin main
```

## 3. GitHub Pages を有効にする
GitHub で次を開きます。

- Repository
- Settings
- Pages
- Build and deployment
- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`
- Save

数分待つと公開されます。

## 4. 公開 URL

### Project Pages の URL
```text
https://<GitHubユーザー名>.github.io/event-manager-a/
```

### User Pages にしたい場合
Repository 名を `<GitHubユーザー名>.github.io` にすると、URL は次になります。
```text
https://<GitHubユーザー名>.github.io/
```

## 5. 最低限の動作確認
- 一覧にイベントが出る
- 「＋ 新しいイベントを作る」で追加できる
- 詳細画面で内容が見える
- 編集が反映される
- 削除確認後に消える
- 参加 / キャンセルで人数が変わる
- F5 しても内容が残る
